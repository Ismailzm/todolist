 Task 2 – Python To-Do List Application

This project is developed as a part of the Internship Task.  
It is a simple and efficient **command-line based To-Do List Application** made using Python.



 Features

- Add new tasks  
- View all tasks  
- Remove any task  
- Tasks are stored in `tasks.txt`  
- Persistent storage (tasks remain even after closing the program)



Project Structure

Task2/
│── todo.py # Main Python application
│── tasks.txt # File where tasks are stored automatically
│── README.md # Project documentation
